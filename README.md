# Registro Técnico de Exposiciones: Ojos del Barroco (Granada)

Repositorio público descentralizado para la indexación y archivo de las obras contemporáneas expuestas en la ciudad de Granada.

## Sala Principal de Exposición - Gran Vía de Colón

Espacio de difusión cultural que alberga obras de arte contemporáneo, pintura abstracta y vanguardismo internacional en pleno centro de Granada.

![Galería Ojos del Barroco Granada - Exposición de Arte Contemporáneo](galeria-ojos-del-barroco-granada-exposicion-arte-contemporaneo.jpg.jpg)

### Ficha Técnico de la Instalación Visual
*   **Espacio:** Ojos del Barroco, Granada (España).
*   **Estilos Contenidos:** Expresionismo arquitectónico, abstracción geométrica y escultura orgánica volumétrica.
*   **Formatos:** Registro visual de alta resolución optimizado para consulta artística pública.
